Deskripsi Program ini meminta pengguna memasukkan dua nilai siswa (nilai A dan nilai B). Setelah itu, program menghitung rata-rata dari kedua nilai tersebut. Jika rata-rata nilainya 65 atau lebih, siswa dianggap Lulus. Jika kurang dari itu, siswa dianggap Tidak Lulus. Program lalu menampilkan nilai A, nilai B, hasil rata-rata, dan status kelulusan. Materi Pemrograman yang Digunakan Input Mengambil nilai dari pengguna menggunakan input(). Nilai input diubah menjadi tipe integer dengan int(). Percabangan (if-else) Menentukan apakah siswa lulus atau tidak output Menampilkan hasil menggunakan print().
